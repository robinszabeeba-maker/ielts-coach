## 🚀 详细步骤（跟着做就行）

### 步骤1：登录Streamlit Cloud
1. 打开浏览器，访问：https://streamlit.io/cloud
2. 点击右上角 "Sign in"
3. 选择 "Sign in with GitHub"
4. 授权Streamlit访问你的GitHub账号

### 步骤2：找到你的应用
登录后，你会看到类似这样的界面：
```
[你的头像] Dashboard
└── Your apps
    └── ielts-coach (robinszabeeba-maker/ielts-coach)
```

**点击 "ielts-coach"** 进入应用管理页面。

### 步骤3：进入设置页面
在应用页面，你会看到：
- 应用预览
- 访问链接：`https://ielts-coach-robinszabeeba-maker.streamlit.app`
- 一些按钮

**找到并点击 "Settings" 按钮**（通常在右上角或应用信息下方）。

### 步骤4：更改主文件路径
在Settings页面，找到 "Main file path" 设置：

**当前可能是**：`ielts_coach_optimized.py` 或 `ielts_coach.py`

**需要改为**：`ielts_coach_theme.py`

**操作**：
1. 点击编辑框
2. 删除旧的文件名
3. 输入：`ielts_coach_theme.py`
4. 确保拼写完全正确

### 步骤5：保存并重新部署
1. 滚动到页面底部
2. 点击 "Save" 或 "Update" 按钮
3. 系统会自动重新部署
4. 等待1-2分钟，看到 "Deployed successfully" 提示

### 步骤6：访问新应用
部署完成后：
1. 点击应用页面上的链接
2. 或直接访问：`https://ielts-coach-robinszabeeba-maker.streamlit.app`
3. 你会看到全新的主题周版本界面

## 🆘 常见问题解决

### Q1：找不到Settings按钮？
A：尝试这些位置：
- 应用卡片右上角的 "⋮"（三个点）菜单
- 应用详情页面的选项卡："App"、"Settings"、"Logs"
- 页面右上角的齿轮图标 ⚙️

### Q2：保存后显示错误？
A：检查：
1. 文件名拼写：`ielts_coach_theme.py`（注意是下划线，不是空格）
2. GitHub仓库是否有这个文件（应该有，我已推送）
3. 错误信息是什么，截图发给我

### Q3：应用无法访问？
A：等待2-3分钟再试，Streamlit Cloud部署需要时间。

### Q4：还是旧版界面？
A：可能是浏览器缓存：
1. 按 Ctrl+F5（Windows）或 Cmd+Shift+R（Mac）强制刷新
2. 或打开无痕窗口访问
