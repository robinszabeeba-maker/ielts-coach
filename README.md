# 雅思备考智能教练

专为Leon定制的雅思备考智能学习工具，根据当前英语水平（四级440+）和目标分数（7.5分）生成个性化学习计划。

## 核心功能

1. **智能学习计划**：基于你的水平和目标，生成12周每日学习任务
2. **每日打卡**：完成听说读写4项任务，简单打卡记录
3. **进度可视化**：日历打卡视图 + 能力雷达图
4. **自适应调整**：每周根据完成情况调整下周计划

## 技术栈

- Python 3.9+
- Streamlit (Web界面)
- SQLite (本地数据库)
- Plotly (数据可视化)

## 使用流程

### 第一步：初始配置
1. 设置目标分数（7.5）
2. 选择当前水平（四级440+）
3. 设置每日学习时间（2小时）
4. 确认背单词工具（墨墨背单词）

### 第二步：每日学习
1. 打开应用，查看今日4项任务
2. 点击资源链接学习
3. 返回应用，点击"标记完成"打卡
4. 查看日历，确认今日全绿✅

### 第三步：每周回顾
1. 查看本周日历完成情况
2. 更新能力评估（可选）
3. 系统自动调整下周计划难度
